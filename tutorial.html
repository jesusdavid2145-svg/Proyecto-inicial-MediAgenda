Tutorial de uso de MediAgenda

Guía básica para aprender a utilizar las funciones principales del sistema

Índice del tutorial

  1. Crear una cuenta
  2. Iniciar sesión
  3. Agendar una cita médica
  4. Gestionar citas
  5. Consultar notificaciones
  6. Consultar historial clínico autorizado

1. Crear una cuenta

El usuario debe ingresar sus datos personales, correo, contraseña y aceptar los términos para completar el registro.

2. Iniciar sesión

El usuario debe escribir su correo y contraseña para acceder a las funciones disponibles según su rol.

3. Agendar una cita médica

El paciente selecciona especialidad, médico, sede, modalidad, fecha y hora disponible para confirmar la cita.

4. Gestionar citas

El usuario puede consultar, modificar, cancelar o visualizar el detalle de sus citas médicas.

5. Consultar notificaciones

Las notificaciones permiten revisar confirmaciones, recordatorios y alertas relacionadas con las citas.

6. Consultar historial clínico autorizado

El usuario autorizado puede consultar información médica básica, antecedentes y datos de contacto de emergencia.
